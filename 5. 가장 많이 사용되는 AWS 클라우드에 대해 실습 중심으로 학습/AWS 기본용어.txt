가상화 - 물리적 컴퓨터 하드웨어를 보다 효율적으로 활용할 수 있도록 새주는 프로세스, 클라우드 컴퓨팅의 기반을 제공하는 기술

가상머신-  VM은 소프트웨어 형식으로 물리적 컴퓨팅을 시뮬레이션하는 가상 환경이다.

스냅샷 - 스냅샷은 마치 사진 찍듯이 특정 시점에 스토리지의 파일 시스템을 포착해 보관하는 기술
        windows OS의 복원 지점과 같이 장애나 데티어 손상 시 스냅샷을 생성한 시점으로 데이터를 복구
        원본 데이터를 그대로 복사, 데이터의 변경이 있기 전까지는 스토리지의 공간을 차지하지 않음
        메타데이터의 복사본에 해당, 생성하는데 오랜 시간이 걸리지 않고, 장애 상황이 발생해도 빠르게 데이터를 복원

데이터 센터 - 수많은 서버들을 한데 모아 네트위크로 연결해 놓은 시설
